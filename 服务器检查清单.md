## 快速检查命令（Linux服务器）

如果您已经可以SSH登录服务器，可以运行以下命令检查：

```bash
# 检查操作系统
cat /etc/os-release

# 检查是否安装Nginx
nginx -v

# 检查是否安装Apache
apache2 -v
# 或
httpd -v

# 检查防火墙状态
sudo ufw status
# 或
sudo firewall-cmd --list-all

# 检查80和443端口
sudo netstat -tuln | grep -E ':(80|443)'
```
