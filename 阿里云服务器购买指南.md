# 阿里云ECS服务器购买指南

## 一、进入购买页面

### 方法1：从控制台购买
1. 登录阿里云控制台：https://ecs.console.aliyun.com/
2. 点击页面上的 **"创建实例"** 蓝色大按钮
3. 进入购买配置页面

### 方法2：直接访问
访问：https://ecs.console.aliyun.com/home#/createInstance

---

## 二、基础配置选择

### 1. 付费模式
- **推荐选择**：`包年包月`（更稳定，适合长期使用）
- 或者：`按量付费`（按小时计费，适合测试）

### 2. 地域选择
- **推荐**：选择离您最近的地域
- 例如：如果您在河南，可以选择 `华北2（北京）` 或 `华东1（杭州）`
- **重要**：地域选择后不能更改，请谨慎选择

### 3. 实例规格（推荐配置）

#### 方案A：经济型（适合小型网站，推荐）
- **实例规格**：选择 `ecs.t6-c1m2.large` 或类似配置
- **vCPU**：2核
- **内存**：4GB
- **价格**：约 200-300元/月

#### 方案B：标准型（适合中型网站）
- **实例规格**：选择 `ecs.c6.large` 或类似配置
- **vCPU**：2核
- **内存**：4GB
- **价格**：约 300-400元/月

#### 方案C：入门型（最低配置，适合测试）
- **实例规格**：选择 `ecs.t6-c1m1.large` 或类似配置
- **vCPU**：1核
- **内存**：2GB
- **价格**：约 100-200元/月

**建议**：选择方案A（2核4G），性价比高，足够运行网站。

---

## 三、镜像选择（操作系统）

### 推荐选择：Ubuntu 22.04 LTS（推荐）

**为什么选择Ubuntu？**
- 免费、稳定
- 社区支持好
- 适合Web服务器
- 安装软件方便

**选择步骤**：
1. 在"镜像"部分，选择 `公共镜像`
2. 操作系统选择：`Ubuntu`
3. 版本选择：`22.04 64位`（最新LTS版本）

**备选方案**：
- `CentOS 7.9 64位`（如果熟悉CentOS）
- `Alibaba Cloud Linux 3`（阿里云优化版本）

---

## 四、存储配置

### 系统盘
- **类型**：`ESSD云盘`（推荐）或 `高效云盘`
- **容量**：`40GB`（最低，足够使用）
- **如果预算允许**：可以选择 `60GB` 或 `80GB`

### 数据盘（可选）
- 初期可以不购买数据盘
- 后续需要可以随时添加

---

## 五、网络配置

### 1. 专有网络VPC
- **选择**：`默认专有网络` 或创建新的VPC
- **交换机**：选择默认或创建新的

### 2. 公网IP（重要！）
- **必须勾选**：`分配公网IPv4地址`
- **带宽计费方式**：
  - `按固定带宽`（推荐，更稳定）
  - `按使用流量`（如果流量不确定）
- **带宽值**：
  - 选择 `3Mbps`（最低，适合小型网站）
  - 或 `5Mbps`（推荐，更流畅）

### 3. 安全组
- **选择**：`新建安全组`
- **规则模板**：选择 `Web服务器Linux`（会自动开放80、443端口）

---

## 六、系统配置

### 1. 登录凭证
- **推荐选择**：`自定义密码`
- **设置密码**：
  - 密码长度：至少8位
  - 包含：大小写字母、数字、特殊字符
  - **重要**：请记住这个密码，用于SSH登录

### 2. 实例名称
- 可以填写：`吉音动漫网站服务器` 或 `jydm-web-server`
- 方便后续识别

### 3. 主机名（可选）
- 可以留空，使用默认

---

## 七、分组设置（可选）

- **资源组**：可以选择默认
- **标签**：可以添加标签方便管理（可选）

---

## 八、确认订单

### 1. 购买时长
- **推荐**：选择 `1个月`（先试用）
- 或选择 `3个月`、`6个月`、`1年`（有折扣）

### 2. 确认配置
检查以下信息：
- ✅ 实例规格（2核4G）
- ✅ 操作系统（Ubuntu 22.04）
- ✅ 系统盘（40GB）
- ✅ 公网IP（已分配）
- ✅ 带宽（3Mbps或5Mbps）
- ✅ 密码（已设置）

### 3. 阅读并同意协议
- 勾选服务协议

### 4. 提交订单
- 点击 **"确认订单"** 或 **"立即购买"**
- 完成支付

---

## 九、购买后操作

### 1. 等待实例创建
- 通常需要 1-3 分钟
- 在控制台可以看到创建进度

### 2. 获取服务器信息
创建完成后，在实例列表中可以看到：
- **实例ID**
- **公网IP**（这就是您的服务器IP地址！）
- **内网IP**
- **状态**：运行中

### 3. 记录重要信息
请记录：
- **公网IP地址**：`________________`
- **登录用户名**：`root`（Ubuntu系统）
- **登录密码**：`________________`（您设置的密码）

---

## 十、购买后立即需要做的事情

### 1. 测试连接
使用SSH工具（如PuTTY）测试能否连接服务器

### 2. 安装Web服务器
连接成功后，安装Nginx：
```bash
sudo apt update
sudo apt install nginx -y
```

### 3. 配置防火墙
```bash
sudo ufw allow 80
sudo ufw allow 443
sudo ufw allow 22
sudo ufw enable
```
